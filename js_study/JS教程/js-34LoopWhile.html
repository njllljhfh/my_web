JavaScript While 循环

只要条件为 true，循环能够一直执行代码块。

While 循环

while 循环会一直循环代码块，只要指定的条件为 true。

语法

      while (条件) {
          要执行的代码块
      }

Do/While 循环

do/while 循环是 while 循环的变体。在检查条件是否为真之前，这种循环会执行一次代码块，然后只要条件为真就会重复循环。

语法

      do {
          要执行的代码块
      }

      while (条件);
