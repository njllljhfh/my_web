JavaScript 逻辑

JavaScript 布尔（逻辑）代表两个值之一：true 或 false

布尔值

通常，在编程中，您会需要只能有两个值之一的数据类型，比如

  • YES / NO
  • ON / OFF
  • TRUE / FALSE

鉴于此，JavaScript 提供一种 布尔 数据类型。它只接受值 true 或 false。

布尔可以是对象

通常 JavaScript 布尔是由字面量创建的原始值：

var x = false

但是布尔也可以通过关键词 new 作为对象来定义：

var y = new Boolean(false)

永远不要将布尔值创建为对象。

无法安全地比较布尔和对象。


不要创建布尔对象。它会拖慢执行速度。
