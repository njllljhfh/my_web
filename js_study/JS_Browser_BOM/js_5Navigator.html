JavaScript Window Navigator

window.navigator 对象包含有关访问者的信息。

Window Navigator

window.navigator 对象可以不带 window 前缀来写。

一些例子：

  • navigator.appName
  • navigator.appCodeName
  • navigator.platform

实例

警告！！！

来自 navigator 对象的信息通常是误导性的，不应该用于检测浏览器版本，因为：

  • 不同浏览器能够使用相同名称
  • 导航数据可被浏览器拥有者更改
  • 某些浏览器会错误标识自身以绕过站点测试
  • 浏览器无法报告发布晚于浏览器的新操作系统
